Java

基本数据类型

整型数据前缀
  • 十进制：50
  • 八进制：062（前缀0）
  • 十六进制：0x32（前缀0x）
默认类型
  • 整数默认为int
  • 小数默认为double（定义float数据，数字后跟上f/F）

变量类型

  • 局部变量：构造函数或块内部声明的变量，需要初始化，否者编译错误。
  • 实例变量：属于类的实例，如果不初始化，直接赋值为默认值。
  • 静态变量或类变量：类变量是在类中用static关键字声明的变量，属于类但不是实例。
  • 参数变量：参数是方法或者构造函数声明中的变量。

命名格式

  • 驼峰法
    • 首字母小写：局部变量、实例变量（成员变量）、参数
    • 首字母大写：类名
  • 全部大写：常量

修饰符

访问控制修饰符

default（默认修饰符）

默认是同一包内的所有类都可以访问

private（私有访问）

private声明的变量、方法、构造方法只能被所属类访问，并不能对类和接口进行声明

public（公有访问）

public声明的类、方法、构造方法和接口能够被其他类访问

protected（受保护的访问）

protected 可以修饰数据成员，构造方法，方法，不能修饰类（内部类除外）（接口和接口的成员方法和变量均不可）

访问控制和继承
  • 父类public子类public
  • 父类protected子类protected、public
  • 父类private不能继承

非访问修饰符

static

  • 静态变量：类变量，局部变量不可声明为静态变量。无论实例多少份对象，类变量 只有一份拷贝

  • 静态方法：独立于对象的静态方法，静态方法不能使用类的非静态变量。

对类变量和方法的访问可以直接使用 classname.variablename 和 classname.methodname 的方式访问

final

  • final变量：必须指定初始值，常和static一起 声明类常量

  • final方法：final方法可以被子类继承但是不能被重写

  • final类：无法继承

abstract
  • 抽象类：类无法被final和abstract同时修饰，作用未来对该类进行扩充，如果有抽象方法，那么必为抽象类
  • 抽象方法：没有实现，需要子类实现。不能被final和static修饰，继承抽象类的子类必须实现所有抽象方法
synchronized

方法同一时间只能被一个线程访问。可以用四个访问修饰符

transient

该修饰符包含在定义变量的语句中，用来预处理类和变量的数据类型

volatile

String

String创建的对象，是无法修改的

StringBuffer

每次都会对 StringBuffer 对象本身进行操作，而不是生成新的对象

StringBuilder

StringBuilder 的方法不是线程安全的（不能同步访问），但是相较于StringBuffer速度更快

  • StringBuilder sb = new StringBuilder(10)；
    • 对象的大小，初始大小设定
  • sb.append(“456”)；
    • 类似于python向内存添加数据
  • sb.insert(4,”89”);
    • 在索引为4处插入——”45689“
  • sb.delete(2,4)
    • 删除索引[2,4)间的数据——“459”
