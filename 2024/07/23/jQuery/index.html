使用jQuery前必须在html页面中添加一些东西。首先，在页面顶部添加script标签，浏览器会运行script标签所有的JavaScript脚本包括jQuery。

在 script 标签中添加代码 $(document).ready(function() {。 然后在后面（仍在该 script 标签内）用 });;闭合它。

在浏览器加载页面后，你放入此 function 的代码将立即运行。

如果没有 document ready function，代码将在 HTML 页面呈现之前运行，这可能会导致错误。
